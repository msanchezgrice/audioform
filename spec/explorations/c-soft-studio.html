Getting started

Talkform turns any form definition into a live audio interview. Define your fields, embed the widget or call the API, and get structured JSON back.

Install

Add the React package:

npm install @talkform/react

Basic usage

Import the widget and pass your config:

import { AudioformWidget } from '@talkform/react';

<AudioformWidget
  config={myFormConfig}
  onComplete={(result) => console.log(result)}
/>

Configuration

The AudioformConfig object describes the fields you want to capture, the prompt, and validation rules.
