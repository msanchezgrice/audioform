<iframe
  src="https://talkform.ai/embed/customer-feedback"
  width="520"
  height="640"
  frameborder="0"
  allow="microphone; autoplay"
  style="border-radius:28px;border:1px solid #eee"
/>

Works like Typeform or Calendly embeds. The iframe hosts the full widget — audio capture, transcript, variable extraction — and posts the completed result back via postMessage when done. The host page just listens for the result event. Microphone permission is scoped to the iframe origin.

import { TalkformEmbed } from '@talkform/react';

function FeedbackPage() {
  return (
    <TalkformEmbed
      formId="customer-feedback"
      onComplete={(result) => {
        // result is AudioformSessionResult
        console.log(result.fields);
      }}
    />
  );
}

The React component renders the widget inline (no iframe). It manages audio, WebRTC, and field extraction internally. The onComplete callback fires with the full session result when all required fields are captured. Style it with CSS variables or the theme prop.

<!-- Load the script -->
<script src="https://talkform.ai/embed.js"></script>

<!-- Mount point -->
<div id="talkform"></div>

<script>
  Talkform.mount('#talkform', {
    formId: 'customer-feedback',
    onComplete: (result) => {
      // Send to your backend
      fetch('/api/feedback', {
        method: 'POST',
        body: JSON.stringify(result)
      });
    }
  });
</script>

Framework-agnostic. Load the script, call Talkform.mount() with a CSS selector and config. Works in WordPress, Webflow, plain HTML, or any CMS. The script bundles the same React widget internally.
