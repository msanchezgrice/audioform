Getting started

Talkform turns any form definition into a live audio interview. Define your fields in a config object, embed the widget or call the API, and get structured JSON back.

Install

Add the React package to your project:

npm install @talkform/react

Basic usage

Import the widget and pass your config. The widget handles the full audio interview lifecycle — microphone permissions, real-time transcription, field extraction, and result export.

import { AudioformWidget } from '@talkform/react';

<AudioformWidget
  config={myFormConfig}
  onComplete={(result) => console.log(result)}
/>

Configuration

The AudioformConfig object describes the fields you want to capture, the prompt the host will follow, and any validation rules. See the full configuration reference.
